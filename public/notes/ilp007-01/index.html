Programação Orientada a Objetos - Aula 01

February 11, 2025

ILP007 - Programação Orientada a Objetos

Objetivo

Implementar softwares com uso de linguagem de programação orientada a objetos.

Fundamentos de Orientação a Objetos Abstração em POO Interfaces em POO

Conteúdo

O conteúdo da primeira aula foi basicamente o que já havia tomado nota em Engenharia de Software III - Aula 01 e nas demais anotações ligadas à esta.

Códigos

Estou utilizando o Github para armazenar os códigos desta matéria e, para aumentar o meu conhecimento de git, estou fazendo a partir do repositório do professor.

Para isso, realizei um fork do repositório maromo71/poonoite, e configurei para que ele seja upstream no meu repositório de origem (origin) armazenado em GabrielCoelho/ILP007_codes

Assim, mantenho os códigos realizados pelo professor em sala na branch:main, e os meus códigos numa branch separada, branch:my_codes.
